## ‚ñ∂Ô∏è How to Run the Project

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sahilghoghari07/React-Learning.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd React-Learning/json-api
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start JSON server:**
   ```bash
   npx json-server --watch db.json --port 5000
   ```
   The app will run at:
   http://localhost:5000

5. **Start Vite development server:**
   ```bash
   npm run dev
   ```
   The app will run at:
   http://localhost:5173
